The se_colors Namespace




This namespace contains everything related to the ansi color codes packed in nested namespaces for readability and use. The nested namespaces that are used in this class to organize the colors are:
  • formats
    - This namespace holds format codes such as bold or dim
  • basic_colors
    - This namespace holds the ansi codes for the non-bold/bright variants of colors
    • foreground
      - This nested namespace holds the color codes that only deal with the foreground colors for basic colors
    • background
      - This nested namespace holds the color codes that only deal with the background colors for basic colors
    • combos
      - This nested namespace holds the color codes that only deal with the color combinations for all the basic variants of color codes for both foreground and background.
  • bright_colors
    - This namespace holds the ansi codes for the bold/bright variants of colors
    • foreground
      - This nested namespace holds the color codes that only deal with the foreground colors for bright colors
    • background
      - This nested namespace holds the color codes that only deal with the background colors for bright colors
    • combos
      - This nested namespace holds the color codes that only deal with the color combinations for all the bright variants of color codes for both foreground and background.
  • Tag
    - This namespace holds functions that wrap a message or word(s) in the specified color, beginning with the color code before the input and ending with the ansi reset code
